## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
# mitraM

MitraM — Gujarati community accounting and ledger management platform. This repository contains a lightweight React + Vite frontend and a small Express-based dev server used for demos and local testing.

Quick highlights:
- Multi-year ledger and member distribution engine
- Local first: simple JSON DB and low-polling sync
- Small footprint for low-end servers and periodic usage
